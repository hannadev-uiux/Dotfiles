# 🌸 Seus Atalhos do Terminal

> Digite **`helpme`** a qualquer momento pra ver esta tela. Sai com **`q`**.

---

## 🤖 Claude Code

| Você digita | O que faz |
|---|---|
| `vibe` | Abre o Claude Code **com o fundo roxo** (assim você sabe que é o terminal dele) |
| `claude` | A mesma coisa que `vibe` |

Quando o fundo está **roxo** e a aba diz **🤖 CLAUDE CODE**, você está conversando
com o Claude Code. Quando o fundo volta ao normal, é um terminal comum.

## ▶️ Rodar um projeto Node.js

| Você digita | O que faz |
|---|---|
| `deps` | Instala as dependências do projeto (o `npm install`) |
| `run` | Abre um **menu** com os scripts do `package.json` — escolha com as **setas** e Enter |
| `dev` | Roda direto o script `dev` (servidor de desenvolvimento) |

Não precisa decorar os comandos do projeto: digite `run`, use ↑ ↓ pra escolher, Enter.

---

## ⌨️ Atalhos com as SETAS

| Tecla | O que faz |
|---|---|
| **↑ / ↓** | Procura comandos que você já digitou (digite o começo e use ↑) |
| **→** (seta direita) | Aceita a sugestão em cinza |
| **Option + ← / →** | Pula uma palavra por vez |
| **Option + Delete** | Apaga a palavra anterior inteira |
| **Tab** | Completa nomes de pasta/arquivo (use as setas no menu) |
| **Home / End** | Vai pro começo / fim da linha |

---

## 📁 Andar pelas pastas

| Você digita | O que faz |
|---|---|
| `ls` | Lista o que tem na pasta |
| `cd nome-da-pasta` | Entra numa pasta (ou só digite o nome dela) |
| `..` | Volta uma pasta |
| `proj` | Vai direto pra sua pasta de projetos |
| `o` | Abre a pasta atual no Finder |
| `c` | Limpa a tela |

A **prompt** (a linha colorida) sempre mostra em qual pasta você está. Se ficar
perdida, olhe pra ela.

## 💾 Salvar no Git (simples)

| Você digita | O que faz |
|---|---|
| `status` | Mostra o que mudou |
| `salvar "mensagem"` | Salva suas mudanças (ex: `salvar "ajustei a cor do botão"`) |
| `enviar` | Manda pro GitHub |
| `baixar` | Traz as novidades do GitHub |

## ⌨️ Aprender o Vim/editor

| Você digita | O que faz |
|---|---|
| `vimhelp` | Abre a cheatsheet do Vim/Neovim (atalhos do editor) |
| `nvim arquivo` | Abre um arquivo no editor (LazyVim) |

O editor é o **LazyVim**, com o menu em português: dentro dele, aperte a
**barra de espaço** e espere — aparece um menu explicando o que cada tecla faz.

## 🆘 Se travar

- Apertou algo e o terminal "prendeu"? Tente **Control + C** pra cancelar.
- Quer começar do zero? Digite `c` pra limpar a tela.
- Esqueceu um atalho? **`helpme`**. 💛
